# ⚡ Arcs Elec

**Arcs Elec** est un site web vitrine moderne dédié à une entreprise d'électricité basée aux Arcs. Conçue pour être esthétique et interactive, cette interface entièrement "responsive" offre une expérience utilisateur fluide sur tous les appareils.

---

## ✨ Fonctionnalités

- **Interface Dynamique** : Effets de scroll personnalisés avec un logo qui s'adapte (sticky) lors de la navigation.
- **Menu Burger Mobile** : Navigation optimisée pour les smartphones.
- **Galerie Interactive** : Affichage des réalisations avec un système de "modal" (cliquer sur une image pour l'agrandir).
- **Contact Facilité** : 
    - Bouton d'adresse qui copie l'adresse dans le presse-papier au clic.
    - Liens directs vers le numéro de téléphone et l'adresse email.
- **Expérience Utilisateur** : Bouton de retour en haut de page fluide.
