# ⚡ Arcs Elec

**Arcs Elec** est un projet web moderne et vitrine pour une entreprise d'électricité. Le site est conçu pour être à la fois esthétique, interactif et entièrement "responsive", offrant une expérience utilisateur fluide sur tous les appareils.

---

## ✨ Fonctionnalités

- **Interface Dynamique** : Effets de scroll personnalisés avec un logo qui s'adapte (sticky) lors de la navigation.
- **Menu Burger Mobile** : Navigation optimisée pour les smartphones.
- **Galerie Interactive** : Affichage des réalisations avec un système de "modal" (cliquer sur une image pour l'agrandir).
- **Cartographie Intégrée** : Utilisation de **Leaflet** pour afficher la localisation de l'entreprise.
- **Contact Facilité** : 
    - Bouton d'adresse intelligent qui copie l'adresse dans le presse-papier au clic.
    - Liens directs vers le numéro de téléphone et l'adresse email.
- **Expérience Utilisateur** : Bouton de retour en haut de page fluide.

---

## 🛠️ Technologies Utilisées

- **HTML5** : Structure sémantique du site.
- **CSS3** : Design personnalisé, système de grille (Grid) complexe pour la galerie, et animations.
- **JavaScript (Vanilla)** : Gestion du DOM, des événements (copy-to-clipboard, burger menu, modal) et de l'observateur d'intersection (Intersection Observer API).
- **Leaflet.js** : Pour la carte interactive.
- **Google Fonts & Font Awesome** : Iconographie et typographies (Gotham).

---

## 📂 Structure du Projet

- `index.html` : Contenu principal et structure de la page.
- `style.css` : Styles globaux, gestion du responsive et de la grille d'images.
- `script.js` : Toute la logique interactive (copie d'adresse, menu, effets de scroll).
- `source/` : Dossier contenant les images et les assets (favicon, logos).
- `police/` : Polices de caractères locales (Gotham Black, Gotham Medium).
